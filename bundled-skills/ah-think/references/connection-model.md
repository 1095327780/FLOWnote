# 联结模型（推测模型）

## 底层原理

联结模型定义“输入空间 -> 输出空间”的映射关系。

它不是概念定义本身，而是概念之间的关系结构（函数、规则、约束、机制）。

## 前置条件

- 输入变量的判别模型已建立。
- 输出变量的定义清晰可验证。
- 有最小可用样本或机制依据支持映射。

## 详细操作流程

1. **定义输入空间**：已知变量、取值范围、上下文条件。
2. **定义输出空间**：希望推测的目标结果。
3. **选择关系形态**：因果/相关/规则约束。
4. **建立映射**：公式、决策规则或结构关系。
5. **实例推断**：把具体对象映射为输入变量，执行推测。
6. **结果回译**：把输出变量翻译回可行动结论。
7. **反例检验**：找至少 1 个失效情境修正边界。

## 反模式

- 输入变量本身未定义清楚就开始推测。
- 相关关系直接当因果关系。
- 不声明适用条件，导致泛化过度。

## 质量检查

- 输入变量是否可观测、可判别？
- 映射关系是否可复现？
- 是否明确了失效边界？

## 输出模板

- `input_space`
- `output_space`
- `mapping_rule`
- `boundary_conditions`
- `prediction_example`
