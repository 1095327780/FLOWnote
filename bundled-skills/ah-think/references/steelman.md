# 钢人化反方 (Steelman)

## 底层原理

钢人化要求先构建“最强反方论证”，再检验自己观点。

它的目的不是自我否定，而是提升结论稳健性与边界清晰度。

## 适用场景

- 观点争议大、影响范围广。
- 决策可能受立场偏见影响。
- 制卡前需验证观点可辩护性。

## 不适用 / 降级条件

- 时间极紧且决策可逆，做简版反方校验即可。

## 详细操作流程

1. **定义己方主张**：一句话 + 当前置信度。
2. **重建反方最强版本**：补齐其最佳证据、最佳逻辑、最佳场景。
3. **公平比较**：对比双方在不同条件下的优势区间。
4. **定位分歧根因**：是事实分歧、价值权重分歧，还是时间尺度分歧。
5. **修订己方结论**：收窄边界、增加条件、或改变策略。
6. **输出可辩护版本**：写出“我主张 X，但在 Y 条件下采用 Z”。

## 与 FLOW 的结合

- `ah-read`：防止只摘录同温层观点。
- `ah-card`：产出“主张 + 最强反方 + 边界修订”卡。
- `ah-review`：复盘是否忽视关键反证导致误判。

## 常见误用与纠偏

- 误用：构建稻草人反方，轻易打败。
  纠偏：反方版本必须让自己感到真实压力。
- 误用：为了平衡而模糊结论。
  纠偏：钢人化后仍需给可执行立场。

## 质量检查

- 是否真的使用了“最强”反方证据？
- 结论边界是否更清晰而非更含糊？
- 是否明确何时坚持原判断、何时切换？
