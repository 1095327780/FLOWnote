# 苏格拉底式提问 (Socratic Questioning)

> "我唯一知道的就是我什么都不知道。"
> — 苏格拉底

---

## 背景

苏格拉底（公元前470-399年）是古希腊哲学家，被誉为西方哲学的奠基人之一。他有一种独特的教学方法：不直接给出答案，而是通过一系列精心设计的问题，引导对方自己发现真理。

这种方法被称为"苏格拉底式提问"或"苏格拉底方法"（Socratic Method）。它的核心不是为了"赢"辩论，而是为了**通过提问来揭示隐藏的假设、检验逻辑、澄清概念，最终接近真理**。

**核心洞见**：很多我们"知道"的东西，其实经不起追问。苏格拉底式提问帮助你发现自己思维中的漏洞和盲点。

---

## 核心原理

苏格拉底式提问基于几个假设：

1. **最好的学习是自己发现**：被告知的知识远不如自己发现的知识深刻
2. **提问比回答更有力**：正确的问题能打开思维，而答案往往关闭思维
3. **承认无知是智慧的开始**：只有意识到自己不知道，才能真正开始学习

---

## 六类核心问题

理查德·保罗（Richard Paul）将苏格拉底式提问系统化为六类问题。每类问题有不同的作用。

### 1. 澄清性问题 (Clarification)

**目的**：确保你和对方说的是同一件事；把模糊的概念变清晰

**典型问题**：
- "你说的XX具体是什么意思？"
- "能举一个例子吗？"
- "你能用另一种方式说一遍吗？"
- "当你说XX的时候，你指的是YY还是ZZ？"

**AI引导**：
```
这个概念有点模糊，我们来澄清一下：

「XX」具体指什么？
- 能给一个具体的例子吗？
- 它和YY有什么区别？
```

### 2. 探究假设 (Probing Assumptions)

**目的**：挖出隐藏的前提假设，检验它们是否成立

**典型问题**：
- "这个观点的前提假设是什么？"
- "你为什么这样假设？"
- "这个假设一定成立吗？"
- "如果这个假设不成立呢？"

**AI引导**：
```
让我们检查一下这个观点的假设：

「XX」这个观点成立，需要什么前提条件？
- 这些条件总是满足吗？
- 有没有例外情况？
```

### 3. 探究理由和证据 (Probing Reasons and Evidence)

**目的**：检验支持观点的理由是否充分

**典型问题**：
- "你为什么这样认为？"
- "有什么证据支持这个观点？"
- "这个理由足够吗？"
- "还有其他证据吗？"

**AI引导**：
```
我们来看看支持这个观点的理由：

你为什么相信「XX」？
- 有什么证据？
- 这个证据可靠吗？
- 有没有反面的证据？
```

### 4. 探究视角和观点 (Questioning Viewpoints and Perspectives)

**目的**：看到其他可能的视角，避免片面

**典型问题**：
- "还有没有其他看法？"
- "从YY的角度看，会怎样？"
- "如果换一个人，他会怎么想？"
- "反对这个观点的人会怎么说？"

**AI引导**：
```
让我们换个角度看：

如果有人不同意「XX」，他可能会怎么说？
- 他们的理由是什么？
- 他们看到了什么你没看到的？
```

### 5. 探究影响和后果 (Probing Implications and Consequences)

**目的**：追溯观点的逻辑结论，看看是否合理

**典型问题**：
- "如果这是对的，会意味着什么？"
- "这会导致什么后果？"
- "这和YY是否矛盾？"
- "按这个逻辑推下去，结论是什么？"

**AI引导**：
```
如果「XX」是对的：

- 这意味着什么？
- 会导致什么后果？
- 有没有什么不合理的结论？
```

### 6. 关于问题的问题 (Questions about the Question)

**目的**：反思问题本身，确保问对了问题

**典型问题**：
- "这真的是正确的问题吗？"
- "这个问题为什么重要？"
- "还有什么问题我们应该问？"
- "问这个问题的目的是什么？"

**AI引导**：
```
让我们退一步想想：

我们为什么要讨论「XX」？
- 这是真正重要的问题吗？
- 有没有更根本的问题应该先问？
```

## 阅读笔记中的应用

### 场景1：遇到一个论断/观点

```
书中说「XX导致YY」。

🔍 苏格拉底检验：

1. [澄清] XX具体指什么？YY具体指什么？
2. [假设] 这个因果关系的假设是什么？
3. [证据] 作者给了什么证据？证据充分吗？
4. [视角] 有没有人持不同观点？
5. [后果] 如果这是对的，意味着什么？
```

### 场景2：自己形成了一个洞见

```
读完这段，你的洞见是「XX」。

🔍 自我检验：

1. [澄清] 你说的XX，能再解释清楚一点吗？
2. [假设] 这个洞见基于什么假设？
3. [证据] 有什么支持这个想法？
4. [反面] 什么情况下这个洞见不成立？
```

### 场景3：深度理解一个概念

```
你想深入理解「XX」这个概念。

🔍 苏格拉底探索：

1. [定义] XX的本质是什么？
2. [边界] 什么是XX，什么不是XX？
3. [关系] XX和YY有什么关系？
4. [应用] XX在什么场景下最有价值？
```

## 引导技巧

### 递进式提问

从简单到深入，一步步引导：

```
1. "你说的XX是什么意思？"（澄清）
2. "为什么你这样认为？"（理由）
3. "这个观点的前提是什么？"（假设）
4. "如果这个前提不成立呢？"（检验）
5. "还有没有其他可能？"（视角）
```

### 温和但坚定

苏格拉底式提问不是"审问"，而是"共同探索"：

- ✅ "我想理解一下，你说的XX是指…？"
- ❌ "你这个说法有问题，XX怎么可能…"

### 保持好奇心

真正的苏格拉底精神是**求知**，不是**证明别人错**：

- ✅ "有意思，如果反过来想呢？"
- ❌ "你这样想肯定是错的"

## 完整对话示例

**用户**：我读到"早起的人更成功"，感觉挺有道理的。

**AI（苏格拉底式引导）**：

```
这个观点确实很流行。我们来检验一下？

[澄清] 首先，这里的"成功"指什么？事业成就？财富？还是幸福感？

[用户回答]

[假设] 这个观点假设了什么？是不是假设早起和成功之间有因果关系？

[证据] 有什么证据支持这个关系？是相关性还是因果性？

[视角] 有没有晚睡晚起但很成功的人？他们怎么说？

[后果] 如果这是对的，意味着我们应该强迫自己早起吗？即使违背自己的生物钟？

你觉得，经过这些追问，你对这个观点的看法有变化吗？
```

## 注意事项

### ✅ 做到

- 真心好奇，而非审问
- 从简单问题开始
- 给对方（或自己）思考的时间
- 承认自己也可能是错的

### ❌ 避免

- 把提问变成辩论
- 一次问太多问题
- 用问题"攻击"对方
- 追问到对方不舒服

**什么时候用**：
- 读到一个"感觉对但说不清为什么"的观点
- 想深入理解一个概念
- 检验自己或他人的论证
- 想看到问题的多个角度

---

## 与其他模型的配合

- **苏格拉底式提问 + 费曼技巧**：用提问检验是否真正理解
- **苏格拉底式提问 + 逆向思维**：用提问引导看到反面
- **苏格拉底式提问 + 二阶思考**：用提问追问后果

---

## 退出条件

当以下情况出现时，可以结束引导：

- 用户对概念/观点有了更清晰的理解
- 用户发现了之前没想到的角度
- 用户表示"够了""想记录下来"

AI应说：
```
很好，经过这些追问，你对这个观点的理解更清晰了。要把你的思考记录下来吗？
```
