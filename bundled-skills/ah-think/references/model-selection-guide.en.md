# Model Selection Guide

## Choose by Problem Type

- unclear assumptions -> Socratic
- long-term consequence -> Second-order
- root-cause finding -> Five Whys
- concept mastery -> Feynman
- constraint-limited decision -> Circle of Competence

## Session Output Minimum

- one clear conclusion
- one next action
- one validation method

## Escalation

If user needs artifact output (card/project decision), route to `ah-card`/`ah-project` after thinking session.
