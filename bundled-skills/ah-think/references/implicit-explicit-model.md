# 内隐模型与外显模型

## 底层原理

- 外显模型：慢、可解释、可分步控制。
- 内隐模型：快、自动化、依赖高频感知-行动耦合。

学习难点通常不在“知道步骤”，而在“尚未内隐化”。

## 适用场景

- 语言听说、运动、乐器、打字、实时协作。
- 需要把“想得明白”转为“反应得出来”。

## 详细操作流程

1. **任务分型**：当前任务是偏内隐还是偏外显？
2. **外显起步**：先拆步骤，明确判据与反馈。
3. **内隐训练**：高频、短回路、多样化输入，减少过度解释。
4. **自动化迁移**：逐步提高速度与扰动，观察是否保持稳定。
5. **闭环复盘**：记录卡点在“规则不懂”还是“自动化不足”。

## 反模式

- 内隐任务只靠阅读讲义。
- 早期追求速度，跳过正确性反馈。
- 外显规则未清就盲目大量练习。

## 输出模板

- `task_mode`: implicit / explicit / mixed
- `training_protocol`
- `automation_threshold`
- `next_drill`
