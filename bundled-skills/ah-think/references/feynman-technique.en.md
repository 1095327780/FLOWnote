# Feynman Technique

## Purpose

Test real understanding by explaining concepts in simple language.

## Core Moves

1. Pick one concept.
2. Explain it as if to a beginner.
3. Identify gaps or jargon you cannot explain.
4. Refine and explain again with examples.

## Prompt Pattern

- Explain X in plain language.
- Where did the explanation become vague?
- What concrete example can make it clearer?

## Output Requirement

- beginner-friendly explanation
- detected knowledge gaps
- revised explanation
