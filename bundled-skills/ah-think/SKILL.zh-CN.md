# ah-think

提供可插拔的结构化思考会话，用于复杂问题拆解与决策。

## 必须遵守

- Memory 默认最小读取：`STATUS + 当前上下文层`。
- 模板：`assets/templates/思考模板.md` 或 `assets/templates/项目思考模板.md`。
- 输出必须形成可执行结论，不做空泛讨论。

## 可用模型（按需选一个）

- 第一性原理
- 苏格拉底式提问
- 二阶思考
- 反向思考
- 五个为什么
- 费曼技巧
- 能力圈

## 流程

1. 明确问题与边界。
2. 选择一个思维模型。
3. 产出结构化推理：假设、证据、反例、结论。
4. 给出行动项与验证方式。
5. 回写到对应 context memory。

## 输出

- 结论（1-3条）
- 行动项（可执行）
- 可沉淀卡片候选（可选）
