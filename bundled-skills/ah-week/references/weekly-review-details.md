# Weekly Review Details

## Inputs

- Week date range
- Daily notes in range
- Weekly project snapshots

## Metrics

- ideas captured
- ideas processed
- cards created
- key project moves

## Weekly Questions

- What compounded this week?
- What repeated friction should be removed?
- What 1-3 priorities matter next week?

## Output Structure

Use `assets/templates/周回顾模板.md` and ensure:
- trend summary
- unresolved cleanup result
- next week priorities

## Completion

Update `STATUS.md` weekly checkpoint time.
