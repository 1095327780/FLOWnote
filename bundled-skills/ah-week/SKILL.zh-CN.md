# ah-week

周回顾：看趋势、清残留、定下周重点。

## 必须遵守

- Memory 读取模式：跨层摘要（不展开全量历史）。
- 模板：`assets/templates/周回顾模板.md`
- 对未处理想法执行与日回顾一致的三分流规则。

## 流程

1. 汇总本周 daily/card/project 数据。
2. 扫描本周未处理想法并清理。
3. 引导反思：做得好 / 待改进 / 下周约束。
4. 生成并写入周回顾文件。
5. 更新 `STATUS.md` 周期状态。

## 按需读取 References

- 周统计口径、提问模板：`references/weekly-review-details.md`

## 输出

- 周统计摘要。
- 未处理清零结果。
- 下周 1-3 个重点目标。
