# Memory Checklists

## Start Checklist

- [ ] 已读取 `Meta/.ai-memory/STATUS.md`
- [ ] 已识别与当前技能相关的分区条目
- [ ] 如有续跑任务，已给出“继续/新建”选择

## Checkpoint Checklist

- [ ] 任务专属进度已写入
- [ ] `STATUS.md` 已回写
- [ ] 如需交接，已写 `待交接:<skill>`
- [ ] 返回给用户的下一步不超过 3 条

## Error Checklist

- [ ] 状态值不合法时已修正为允许枚举
- [ ] 路径不合法时已替换为 canonical 路径
- [ ] 阻塞项已写为 `阻塞:<原因>`
