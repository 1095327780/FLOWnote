# 链接规范

> 所有 Obsidian 内部链接统一使用 wikilink 语法：`[[路径|显示名]]`

## 主题笔记链接

```markdown
[[02-培养层/主题笔记/📍 心理学|心理学]]
[[02-培养层/主题笔记/📍 思维模型|思维模型]]
```

**注意**：主题笔记位于 `02-培养层/主题笔记/` 文件夹下（文件名以 `📍` 开头）。

## 领域页链接

```markdown
[[03-连接层/🌱 效率工具|效率工具]]
[[03-连接层/🌱 个人成长|个人成长]]
```

**注意**：领域页位于 `03-连接层/` 根目录下（文件名以 `🌱` 开头）。

## 项目链接

```markdown
[[04-创造层/项目/26-02 SnapPlan产品概念设计/📍 项目总览|SnapPlan产品概念设计]]
[[04-创造层/项目/03-知识体系/📍 项目总览|知识体系建设]]
```

**注意**：项目链接应指向**项目总览页**（`📍 项目总览`），而非项目文件夹。
