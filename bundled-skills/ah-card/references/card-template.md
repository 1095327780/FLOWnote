# 永久笔记输出模板

生成永久笔记时使用此模板结构。

## 完整模板

```markdown
---
创建时间: {{YYYY-MM-DD}}
类型: 永久笔记
来源: {{来源书籍/文章}}
状态: 完成
标签: [{{标签1}}, {{标签2}}]
---

# {{标题}}

## 核心观点

{{用户的一句话总结}}

---

## 详细说明

{{用户用自己的话写的详细解释}}

### 为什么重要？

{{用户说明的价值和重要性}}

### 如何应用？

{{用户提供的应用场景和示例}}

---

## 相关笔记

- [[{{相关笔记1}}]] - {{关系说明：支持/反对/补充}}
- [[{{相关笔记2}}]] - {{关系说明}}
- [[{{相关笔记3}}]] - {{关系说明}}

---

## 相关项目

- [[{{项目名}}]] - {{这个知识如何支持该项目}}

---

## 参考来源

- [[{{原书/文献笔记}}]] - {{具体章节/页码}}

---

## 元数据

- **主题**: [[02-培养层/📍 {{主题}}]]
- **创建日期**: {{创建日期}}
- **最后修改**: {{修改日期}}
- **关联数**: {{关联笔记数量}}

---

## 笔记质量检查清单

- [x] **断言式标题**（完整陈述句，表达明确观点）
- [x] 用自己的话表达（不是简单复制）
- [x] 一个笔记只包含一个核心思想
- [x] 建立了至少2-3个相关链接
- [x] 添加了合适的标签
- [x] 说明了实际应用场景
- [x] 链接到相关主题页

---

*本笔记通过 ah-card 技能引导完成*
```

## 使用说明

1. 将 `{{变量}}` 替换为用户在对话中提供的内容
2. 保留用户的原话和表达方式
3. **确保标题是断言式的完整陈述句**（如"地位下降与身体疼痛激活相同脑区"）
4. 确保有至少2-3个双向链接
5. 关系说明要具体（支持/反对/补充哪个观点）
6. 主题页链接格式：`[[📍 心理学主题页|心理学]]`
7. 项目链接应指向项目总览页

## 质量检查

生成笔记前检查：

### 原子化检查
- [ ] 只包含一个核心思想
- [ ] 字数在300-500字左右
- [ ] 没有混入多个概念

### 去语境化检查
- [ ] 用自己的话表达
- [ ] 脱离原文可独立理解
- [ ] 没有"书中提到""作者说"等依赖原文的表达

### 连接检查
- [ ] 有2-3个双向链接
- [ ] 说明了关联关系（支持/反对/补充）
- [ ] 链接到相关主题库

### 来源检查
- [ ] 标注了来源（书名/文章）
- [ ] 标注了具体位置（章节/页码）
