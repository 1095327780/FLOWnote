# Card Workflow Details

## Candidate Selection

Pick one atomic claim per card.

Checklist:
- single claim
- independent meaning
- practical value

## Title Rules

- Use assertion sentence
- Avoid vague noun phrases

## Body Composition

1. claim
2. reasoning
3. application
4. links
5. source/backlinks

## Link Rules

- At least one source backlink required
- Prefer 2+ related note links when available

## Mount Target

Use `card_target` gate before saving.
Default: topic page.

## Quality Gate

Before finalize:
- not paraphrase-only fluff
- no multi-topic mixing
- source trace present
