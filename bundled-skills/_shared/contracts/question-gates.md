# Flow Contract v2 - Question Gates

Use the built-in `question` tool only at key decision points.

## Gate List

- `capture_destination`
  - Choose capture target: daily note / project log / standalone capture
- `review_triage`
  - Choose action for each item: action / card / archive
- `think_insertion`
  - Decide whether to start a structured thinking session
- `card_target`
  - Choose where to mount card: topic / domain / project
- `project_scaffold`
  - Choose scaffold level: full layered structure / lightweight

## Interaction Rules

- Max 3 key questions per run.
- If unanswered, use default and state the chosen default.
- Do not ask low-impact formatting questions.
