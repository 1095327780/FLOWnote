# Index Scan Rules

## 扫描对象

1. 永久笔记（核心知识节点）。
2. 主题页（📍，优先位于 `02-培养层/`，兼容 `02-培养层/永久笔记/`）。
3. 领域页（🌱，位于 `03-连接层/`）。
4. 项目总览（📍 项目总览）与关键项目链接。

## 最小扫描字段

- 页面路径
- 页面类型（永久/主题/领域/项目）
- 出链数量
- 入链数量
- 最近变更时间（可选）

## 分类规则

- 未被任何主题页/领域页引用的永久笔记 -> 候选孤立笔记。
- 主题页中只有列表无分组逻辑 -> 结构薄弱候选。
- 领域页缺少当前项目链接 -> 行动支撑断层候选。

## 增量扫描建议

- 仅扫描最近新增或最近更新页面。
- 若发现异常密集，再升级全量扫描。
