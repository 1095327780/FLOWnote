# Path Conventions

## 目录基线（按当前 FLOW 知识库）

- `01-捕获层/`
- `02-培养层/`
  - `结构笔记/`
  - `文献笔记/`
  - `永久笔记/`
- `03-连接层/`
- `04-创造层/`
  - `Projects/`
  - `Archives/`

## 页面约定

- 主题页（培养层，主路径）：`02-培养层/📍*.md`
- 主题页（兼容旧路径）：`02-培养层/永久笔记/📍*.md`
- 领域页（连接层）：`03-连接层/🌱*.md`
- 项目总览：`04-创造层/Projects/**/📍 项目总览.md`

## 使用说明

- 若库内命名与上述约定不一致，优先做路径映射再执行同步。
- `sync-card` 模式检索主题页时优先主路径，再回退兼容路径。
- `sync-project` 模式默认检索 `03-连接层/🌱*.md` 作为领域页同步目标。
