# Page Creation Rules

## 何时建议新建主题页（📍）

- 同类永久笔记累计到可形成稳定子主题。
- 现有主题页结构过载，新增链接难以定位。

## 何时建议新建领域页（🌱）

- 出现新的长期行动方向或项目簇。
- 需要把多个主题知识聚合到同一行动语境。

## 建议格式

- 建议页面名：`📍 xxx` 或 `🌱 xxx`
- 初始分组：2-4 个分组即可，不做过度设计。
- 首批链接：优先放 3-8 条高价值节点。

## 最小骨架（用于后续稳定同步）

### 主题页（📍）

- 必须包含：`## 📚 核心知识`
- 推荐至少一个 `###` 子分节（如 `### 新增卡片`）

### 领域页（🌱）

- 必须包含：`## 🧠 核心支撑知识` + `### 关键永久笔记`
- 必须包含：`## 🚀 关联项目` + `### 活跃项目`

## 禁止项

- 为单条笔记创建新页面。
- 页面命名与现有页面高度重复。
- 页面创建后无首批内容。
