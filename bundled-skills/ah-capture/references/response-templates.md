# ah-capture 交互文案与输出模板

## 1) 单条写入确认

```text
已记录到今日日记。
- 动作：记录（必写）{{可选：+新增任务/+标记完成/+明日计划}}
- 摘要：{{单段条目摘要}}
- 链接：{{无链接 / 已解析 N 个 / 解析失败 N 个（原始链接已保留）}}
- 位置：01-捕获层/每日笔记/{{YYYY-MM-DD}}.md#{{区块名}}
```

## 2) 批量写入确认

```text
本轮捕获已完成。
- 记录（必写）：{{n}} 条
- 新任务：{{n}} 条
- 标记完成：{{n}} 条
- 明日计划：{{n}} 条
- 链接解析：成功 {{n}} 个，失败 {{n}} 个（原始链接均已保留）
- 文件：01-捕获层/每日笔记/{{YYYY-MM-DD}}.md
```

## 3) 歧义追问（只问一句）

```text
这是要标记任务完成、作为今日任务，还是放到明日计划？
```

## 4) 任务未命中追问

```text
没找到对应任务，要把这条作为已完成记录写到“记录区”吗？
```

## 5) 链接解析失败提示（可选）

```text
链接暂未解析成功，已按原文保留原始链接并完成记录。
```

## 6) 结束追问

```text
还有吗？
```

## 7) 交互模式结束汇总

```text
本轮捕获结束：记录（必写）{{n}} 条，新增任务 {{n}} 条，完成 {{n}} 条，明日计划 {{n}} 条，链接解析成功 {{n}} 个、失败 {{n}} 个（原始链接均已保留）。
如需晚间统一收口，可使用 /ah-review。
```
