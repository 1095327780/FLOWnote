# Archive Playbook

## Preconditions

- Target project exists
- Owner confirms archive action

## Closeout Checklist

- summarize outcomes
- list reusable learnings
- note unresolved follow-ups

## Archive Move

- move to configured archive location (if exists in vault conventions)
- keep overview file path resolvable

## Index Updates

- update project status in memory
- update relevant domain/project index entries

## Optional Hand-off

- if reusable insights found, suggest `ah-card`
