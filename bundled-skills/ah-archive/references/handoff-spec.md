# Archive Handoff Spec

## Status Write Rules

在 `Meta/.ai-memory/STATUS.md` 中：

- 项目分区：
  - 归档完成 -> `已完成`
  - 归档未完成 -> `阻塞:<原因>`
- 卡片笔记分区：
  - 有洞见待转化 -> `待交接:ah-card`

## Call Rules

- 进入协议：`Read ../ah-memory/SKILL.md`
- 经验转化：`Read ../ah-card/SKILL.md`
- 索引维护建议：返回建议 `Read ../ah-index/SKILL.md`（可选）

## Return Rules

返回必须包含：

1. 归档动作结果（迁移/更新/失败）。
2. 复盘沉淀结果（洞见数量与类别）。
3. 下一步建议（最多 3 条）。
