# ah-month

月回顾：战略层审视，处理月度残留，制定下月策略。

## 必须遵守

- Memory 读取模式：各层摘要优先。
- 模板：`assets/templates/月回顾模板.md`
- 本月残留想法必须给出处理结论（行动/卡片/归档）。

## 流程

1. 汇总月度知识产出与项目推进趋势。
2. 清理本月未处理想法。
3. 战略反思：继续做/停止做/新启动。
4. 生成并写入月回顾文件。
5. 更新 `STATUS.md` 与 `index.json` 月度状态。

## 按需读取 References

- 战略复盘维度与产出格式：`references/monthly-review-details.md`

## 输出

- 月度关键结果与趋势。
- 残留清理结果。
- 下月策略与重点清单。
