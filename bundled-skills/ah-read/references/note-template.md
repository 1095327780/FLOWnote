# 文献笔记输出模板

## Frontmatter
字段顺序固定：创建时间、类型、来源、作者、状态、评分、标签；日期格式使用 YYYY-MM-DD。

## 正文结构
固定顺序：基本信息 -> 阅读动机 -> 核心洞见 -> 值得记住的 -> 可转化的永久笔记 -> 整体感受。

## 阅读整理进度区块（文献笔记顶部）
在正文开头固定放置“阅读整理进度”，最少包含：总章节、已完成章节、当前章节、下一章节、最近更新时间。

推荐格式：
- 总章节：数字
- 已完成章节：数字/总章节
- 当前章节：章节名 + 行号范围
- 下一章节：章节名 + 行号范围（无则写“已完成”）
- 最近更新时间：YYYY-MM-DD HH:mm

## 洞见规则
每条洞见写触发线索与个人理解，个人理解优先使用用户原话。

## 转化规则
输出可转永久笔记清单时，只写本轮已确认洞见，不补虚构条目。

## 质量检查
检查标题是否断言式、内容是否可追溯、链接是否完整。
